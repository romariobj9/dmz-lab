# dmz-lab / Laboratorio: Implementación de una DMZ Segura con Cisco Packet Tracer

**Descripción del Proyecto**
Este repositorio contiene el informe completo de un laboratorio de redes donde se implementó una Zona Desmilitarizada (DMZ) segura utilizando Cisco Packet Tracer. El laboratorio simula un escenario real donde una organización necesita exponer un servidor web a Internet mientras protege su red interna.

**Objetivos del Laboratorio**
Configurar direccionamiento IP en una topología de red compleja

Implementar NAT estático para exponer servicios internos de forma segura

Diseñar y aplicar Listas de Control de Acceso (ACLs) para controlar el flujo de tráfico

Establecer una DMZ funcional que aísle diferentes zonas de seguridad

Verificar la conectividad y seguridad entre LAN, DMZ y red externa

**Topología Implementada**
Dispositivos Utilizados:
1 Router Cisco ISR (Router_FW) - Actúa como firewall

3 Switches Cisco 2960 - Uno por cada segmento de red

1 PC Interna (LAN) - Usuario interno

1 Servidor Web (DMZ) - Servicio expuesto

1 PC Externa (Internet) - Cliente externo

Segmentos de Red:
LAN Interna: 192.168.1.0/24

DMZ: 192.168.2.0/24

Red Externa: 192.168.3.0/24
